Typography

Inter at a restrained 3-weight scale. Sentence case always. Size encodes hierarchy — weight reinforces it.

UI patterns

Recurring solutions to recurring problems. Each pattern documents the behavior, not just the look.

Pattern 1 — Delta indicator

Always pair icon + color + numeric value + context label. Never color alone.
